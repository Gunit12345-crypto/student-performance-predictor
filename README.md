# 🎓 Student Performance Prediction

This is a beginner-friendly Machine Learning project.

The project predicts whether a student will PASS or FAIL based on exam scores.

## 📌 PASS Criteria

- Overall percentage must be 40% or more
- Student must score at least 30 marks in each subject

## 🚀 Technologies Used

- Python
- pandas
- numpy
- scikit-learn
- streamlit

## 🤖 Algorithms Used

- Logistic Regression
- KNN

## ▶️ Run Project

Install libraries:

pip install -r requirements.txt

Train model:

python3 model_training.py

Run app:

python3 -m streamlit run app.py
